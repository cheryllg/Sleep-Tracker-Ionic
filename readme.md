# Sleep Tracker

- Built using Ionic. 
- Tracks overnight sleep and sleepiness levels during the day

How to Run: 
1. Install the Ionic CLI globally
```
npm install -g @ionic/cli
```
2. Install project dependencies in the sleeptracker repository
```
cd sleeptracker
npm install
```
3. Run using ionic lab
```
## using lab
ionic lab

##using web server
ionic serve
```
